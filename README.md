
# Firework Ultimate

A simple physical simulation of the effect of a firework explosion

## Installation & Usage

**Clone repository**
```bash
git clone https://github.com/p4sttt/lazarus-firework-ultimate.git
cd ./lazarus-firework-ultimate/bin
```

**Run binary file**
```bash
./firework
```
